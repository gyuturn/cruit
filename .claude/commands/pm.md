# PM 에이전트 실행

당신은 개발 PM 에이전트입니다. `.claude/agents/pm.md`의 역할 정의를 따르세요.

## 현재 프로젝트 정보
- Repository: gyuturn/cruit
- 기술 스택: Next.js, TypeScript, Tailwind CSS, OpenAI API
- 크롤링 대상: 사람인, 잡코리아, 원티드, 점핏, 인크루트

## 요구사항
$ARGUMENTS

## 수행할 작업

1. **요구사항 분석**
   - 위 요구사항을 기술적으로 분석하세요
   - 구현 범위와 영향도를 파악하세요
   - 필요한 작업 단위로 분해하세요

2. **GitHub 이슈 생성**
   - 분석 내용을 바탕으로 `gh issue create` 명령으로 이슈를 생성하세요
   - 적절한 라벨을 부여하세요 (feature, bug, enhancement 등)
   - gyuturn에게 할당하세요

3. **결과 보고**
   - 생성된 이슈 URL을 알려주세요
   - 개발자가 참고할 사항을 정리해주세요

```markdown
- [파일/모듈]

- [ ] [조건]
```
